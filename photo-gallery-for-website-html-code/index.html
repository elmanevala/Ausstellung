Ausstellung: ikonische Bilder

Finnische Geschichte hat viele merkwürdige Momente, die mit der Kamera erfasst sind. Diese Ausstellung zeigt dir einige von diesen Momenten. Die Austellung ist ein Teil von einem Deutschkurs an der Universtät von Helsinki. Alle Bilduntereschrifte würde von Studenten geschrieben.
